We are learning HTML elements, such as links, which can be relative or absolute paths. We will also be setting up a basic form. I'm sure there will be other elements to learn, and we are just scratching the surface of what is possible.

Here is a demonstration of a link. We will showcase a relative path link as well as an absolute path link. A relative path link seems to point to another location that is on the same web server and directory as the current location. On the other hand, an absolute URL links to an entirely different location on the web, although technically it could be on the same web server under a different domain name.
